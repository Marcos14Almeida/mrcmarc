Detecting license plates and pedestrians using YOLO

Development Start: 08/2020. End: 12/2020. Project description YOLO (You Only Look Once) is a real-time object detection algorithm that can detect and identify objects in images and videos. It works by dividing an image into a grid of cells and predicting the object class and bounding box for each cell, using a single convolutional neural network. YOLO is known for its speed and accuracy, making it popular for various computer vision applications such as self-driving cars, surveillance, and robotics.
